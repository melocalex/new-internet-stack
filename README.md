# The New Internet Stack

Landing page for **The New Internet Stack** — a curated [CryptoLar](https://www.cryptolar.com.br/) event on the four layers of the internet being rebuilt in 2026: **AI · Bitcoin · Stablecoins · Sovereignty (proteção patrimonial)**.

📍 FounderHaus · Florianópolis &nbsp;·&nbsp; 📅 01 Aug 2026

> A próxima internet não vai pedir licença. Vai ser construída — com ou sem você na sala.

## Highlights

- **WebGL lightning hero** — raw GLSL bolt that runs only while the hero is on screen.
- **Animated title** — the middle word cycles (Internet → Intelligence → Bitcoin → Stablecoins → Sovereignty), with a gold "Stack".
- **"As Camadas" layers shader** — lightweight raw‑WebGL flowing energy bands; the active layer ignites as you scroll.
- **ASCII sovereign portrait** — image → animated ASCII, with a binary "leak" of floating words (freedom, money, privacy…) into the next section.
- **Scroll-reveal system**, **3D keycap buttons**, gold brand treatment, smooth section transitions.
- **PostHog telemetry** hooks — section views, scroll depth, CTA / coupon clicks.

## Stack

React 19 · TypeScript · Vite · Tailwind CSS v4 · raw WebGL / GLSL

## Develop

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build → dist/
npm run preview  # serve the build
```

> Tip: the WebGL effects animate only in a focused, foreground browser tab.

## Telemetry (optional)

Copy `.env.example` → `.env` and set `VITE_POSTHOG_KEY` to enable analytics. Without it, events just log to the console.
